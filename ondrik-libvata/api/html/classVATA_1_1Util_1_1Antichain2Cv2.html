Detailed Description

template<typename Key, typename T>
class VATA::Util::Antichain2Cv2< Key, T >

An implementation of a 2-component antichain.

This class implements a 2-component antichain, i.e., an antichain over the domain Key x T. An antichain is a set of elements from the domain which are pairwise incomparable w.r.t. a given partial order.

The antichain represents an up closure of its elements, i.e., the set of all elements of the domain that are greater than or equal to some element in the antichain. The notion of greater in our case deserves more discussion. We consider a pair (p, P) to be greater than (q, Q), denoted as (q, Q) <= (p, P), if and only if: q => p && Q <= P.

This implementation is targeted for the use in checking language inclusion of a pair of automata A <= B, where Key is a state of A and T is a set of states of B. The partial order is a composition of the partial orders on the two given domains, i.e., for the intended use, the order on A is the identity (or a simulation) and the order is the set inclusion relation (or set inclusion enhanced by simulation).

Checks whether a pair (q, Q) is covered by the antichain.

This method determines whether a pair (q, Q) is covered by the antichain. In particular, it is given a collection of candidates for 'p' (such that p => q) and tries to find in the antichain a pair '(p, P)' such that P <= Q. The '<=' relation on elements of T is given by cmp. If there is such, the method returns true, otherwise it returns false.

Parameters
[in]candidatesCandidates for 'p'
[in]QThe 'Q' in the pair '(q, Q)'
[in]cmpThe '<=' comparison operation on elements of T
Returns
true if '(q, Q)' is covered by the antichain, false otherwise
Note
The relation cmp needs to be complementary to the one used in the refine() method!

Removes all pairs subsumed by (q, Q) from the antichain.

This method removes all pairs (p, P) covered by (q, Q) from the antichain. In particular, it is given a collection of candidates for 'p' (such that p <= q) and removes from the antichain all pairs '(p, P)' such that P => Q. The '=>' relation on elements of T is given by cmp. An optional eraser on the removed elements is executed.

Parameters

[in]QThe 'Q' in the pair '(q, Q)'
[in]cmpThe '=>' comparison operation on elements of T
[in]eraserAn optional eraser for removed elements
Note
The relation cmp needs to be complementary to the one used in the contains() method!
